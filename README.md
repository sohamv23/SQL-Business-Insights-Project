# SQL-Business-Insights-Project
Overview
This project analyzes an online bookstore dataset using MySQL to generate insights on sales, customers, and inventory.

Objectives
Identify top-selling books and genres
Analyze customer purchasing behavior
Calculate total revenue and sales trends
Evaluate stock availability

Dataset
Books – product details
Customers – customer information
Orders – transaction data

Tool Used
MySQL

Key Analysis
Sales and revenue calculation
Top customers and most ordered books
Genre-wise performance
Stock remaining after orders

Skills
JOINs, GROUP BY, HAVING
Aggregations (SUM, COUNT, AVG)
Data filtering & sorting

Conclusion

This project shows how SQL can be used to analyze data and generate business insights for better decision-making.
